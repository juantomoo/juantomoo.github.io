Créditos

Reconocimiento a las herramientas, librerías y recursos de código abierto que hacen posible este sitio.

Tipografía

  • Inter

    Diseñada por Rasmus Andersson. Utilizada para el cuerpo de texto y UI general por su excelente legibilidad.

  • JetBrains Mono

    Diseñada por JetBrains. Utilizada para bloques de código y elementos técnicos.

Iconografía

  • Feather Icons

    Colección de iconos open source simples y hermosos. Base para la iconografía del sitio.

  • SVG Personalizados

    Logotipos y gráficos abstractos diseñados in-house por Machina Development.

Tecnología

  • Vanilla JS

    Sin frameworks pesados. JavaScript puro para máximo rendimiento.

  • CSS Variables & Grid

    Arquitectura CSS moderna sin preprocesadores externos.

Inspiración

El diseño "Cyber Gold" está inspirado en la estética Cyberpunk, el Brutalismo Web y las interfaces de usuario de ciencia ficción (FUI).
